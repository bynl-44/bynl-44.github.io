dva中实现路由跳转的几种方式

发表于 2019-12-20 阅读次数：

在 dva 中,可以使用多种方式实现路由跳转.

  • Link
  • history

Link

Link 是 react-router-dom 提供的组件,可以用来实现路由跳转.

在 dva 中,可以这样使用:

不能在 外使用.

histroy

histroy, 它提供 push() 方法来实现路由跳转.

histroy 可以有多种途径获取.

props 传递

Router(histroy) -> page -> component, 可以通过这种传递 props 的方式, 在 component 中获取 history.

withRouter

withRouter 是 react-router-dom 提供的组件,可以用来获取 history.

在 dva 中,可以这样使用:

context

import PropTypes from 'prop-types';

const Component = ( props,context)=>{
  return (
    <div>
      <button onClick={ ()=>context.router.history.push('/') }>home</button>
    </div>
  )
}
Component.contextTypes = {
  router: PropTypes.object
}
Component.propTypes = {}
